## Dictionaries

- [Cologne Digital Sanskrit Dictionaries](https://www.sanskrit-lexicon.uni-koeln.de/)
  - Notes: A wonderful list of dictionaries available for use in both browser and download form. If you're not sure, start here.

- [Roots, Verb-Forms, and Primary Derivatives](https://web.archive.org/web/20231023132834/http://sanskrit1.ccv.brown.edu/Sanskrit/whitney/index2.html)
  - Notes: Searchable online, this is a really excellent resource.

- [Master Glossary](https://liberalarts.utexas.edu/)
  - Notes: For use with the lessons from the University of Texas.

- [Sanskrit Dictionary](https://web.archive.org/web/20231023132834/http://spokensanskrit.de/index.php)
  - Notes: Many different modes of input and lots of words.

- [A Vedic Concordance](https://sites.harvard.edu/witzel/)
  - Notes: The download does not work for Windows, but it does work for Mac.

- [Dictionnaire Francais-Sanskrit](http://asignoret.free.fr/index.html)
  - Notes: Downloadable. Ne conviennent pas pour Mac!

- [Glossarium Sanscritum](https://books.google.co.jp/books?id=xaMIAAAAQAAJ&printsec=frontcover&redir_esc=y)
  - Notes: There always has to a very old Latin dictionary.

- [A Sanskrit-English Dictionary](https://books.google.co.jp/books?id=MDEOAAAAIAAJ&pg=PT6&redir_esc=y)
  - Notes: With references to the roots in other languages.

- [Practical Sanskrit-English Dictionary](https://archive.org/details/practicalsanskri00apteuoft)
  - Notes: Appendices with different terms. This is a great resource.

- [Another Practical Sanskrit Dictionary](https://dsal.uchicago.edu/dictionaries/macdonell/)
  - Notes: Practical reference dictionary.

- [A Sanskrit Thesaurus](https://web.archive.org/web/20231023132834/http://www.mediafire.com/?32qkjzzymom)
  - Notes: A bit difficult to use.

- [Sanskrit Glossary](https://sanskritdocuments.org/dict/dictall.pdf)
  - Format: PDF
  - Notes: A fairly long list of words.

- [Sanskrit Flash Cards](https://www.yorku.ca/inpar/language/SanskritFlashCards.pdf)
  - Format: PDF
  - Notes: 3200 (!) words in a very well-formatted PDF file.

- [Apte Sanskrit Dictionary](https://www.aa.tufs.ac.jp/~tjun/sktdic/)
  - Notes: Contains a key for Sanskrit character entry.

- [Word Index to the Nighantu and Nirukta](https://sanskritdocuments.org/sanskrit/major_works/)
  - Notes: A somewhat specialized reference for the study of these texts.

- [Sanskrit-French PDF Dictionary](https://web.archive.org/web/20231023132834/http://sanskrit.inria.fr/Dico.pdf)
  - Format: PDF
  - Notes: The file is very easy to handle, so this ought to be a good resource even if you have to translate again from the French.

- [1000 Sentences in Sanskrit](https://sanskritdocuments.org/learning_tools/daily.pdf)
  - Format: PDF
  - Notes: You should already know the script in order to use this.
