- [Hebrew Philosophical Dictionary](https://archive.org/details/AConciseDictionaryOfHebrewPhilosophicalTerms)

## Texts

- [Online Hebrew OT](https://www.hebrewoldtestament.com/index2.htm)
  - Notes: With many other versions as well.

- [Judeopedia](http://www.judeopedia.org/)
  - Format: PDF
  - Notes: A French site with a very convenient side-by-side Bible with Hebrew, French, English, Latin, and Greek. Can also produce PDF files. An excellent site.

- [The Unbound Bible](https://www.biola.edu/talbot/unbound)
  - Notes: Bible texts online in many languages with parallel viewing.

- [Hebrew OT Reader](https://biblewebapp.com/reader/)
  - Notes: Easy to use, useful even for beginners.

- [The Hebrew Bible](https://www.fourmilab.ch/etexts/www/hebrew/Bible/)
  - Notes: Easy navigation, no vowels.

- [Hebrew Bible Online](https://kodesh.snunit.k12.il/i/t/t0.htm)
  - Notes: Book names in Hebrew. Text has vocalizations.

- [Several Hebrew Bibles](https://mechon-mamre.org/index.htm)
  - Notes: The old text in various forms, with some parallel works as well.

- [Biblos Scripture Text](https://biblehub.com/)
  - Notes: Choose a passage to view in a number of languages. Contains an embedded lexicon.

- [Blue Letter Bible](https://www.blueletterbible.org/index.cfm)
  - Notes: Another multilanguage biblical comparison tool.

- [Interlinear Hebrew/English OT](https://www.scripture4all.org/OnlineInterlinear/Hebrew_Index.htm)
  - Notes: Easy to navigate, not easy to download.

- [Aleppo Codex Online](https://web.archive.org/web/20221113182350/http://aleppocodex.org/flashopen.html)
  - Notes: Read the original.

- [HTML Hebrew OT](https://mechon-mamre.org/dlpt.htm)
  - Notes: A zip file; unzip and place all files in the same folder to navigate.

- [Various Hebrew OT links](https://www.bible-researcher.com/links08.html)
  - Notes: Includes audio codices and audio recordings.

- [Papyrus of the 10 Commandments](https://web.archive.org/web/20221113182350/http://faculty.gordon.edu/hu/bi/ted_hildebrandt/OTeSources/02-Exodus/Text/Articles/Burkitt-10Commands-JQR.pdf)
  - Format: PDF
  - Notes: Very useful article with edited Hebrew of the text.

- [Hebrew New Testament](http://khazarzar.skeptik.net/biblia/hnt/richmond/index.htm)
  - Notes: Some older files, requires the djvu viewer.

- [Hebrew Manuscripts](https://papyri.tripod.com/links-2.html#HEBREW)
  - Notes: A collection of links to various Hebrew manuscripts.

- [Dead Sea Scrolls in English](http://www.gnosis.org/library/dss/dss.htm)
  - Notes: Translations of various Gnostic texts.

- [Dead Sea Scrolls Online](https://www.ibiblio.org/expo/deadsea.scrolls.exhibit/intro.html)
  - Notes: View the texts online.

- [Digital Dead Sea Scrolls](http://dss.collections.imj.org.il/)
  - Notes: The real deal.

- [The Great Isaiah Scroll](https://www.ao.net/~fmoeller/qumdir.htm)
  - Notes: In photograph form.

- [Online Talmud](https://www.e-daf.com/)
  - Notes: You'll need to have some background knowledge in order to make meaningful use of this site.

- [Old Talmudic Manuscript](https://daten.digitale-sammlungen.de/~db/bsb00003409/images/index.html)
  - Notes: Hundreds of years old, in fact.

- [Anatomy of a Talmudic Page](http://ghansel.free.fr/WebTalmud/roch.htm)
  - Notes: Written in French; useful for understanding what the Talmud contains.

- [SHEBANQ Text Database of the Hebrew Bible](https://web.archive.org/web/20221113182350/http://shebanq.ancient-data.org/)
  - Notes: So good that Ricky Martin sang about it in 2000: "SHEBANQ! SHEBANQ! Looking up those Hebrew verbs, the verbs..." Anyway, it's an open access and open source text database for the Hebrew bible. If you're serious about the Hebrew Bible and computational lexical resources, this is for you.
