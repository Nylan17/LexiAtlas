## Scope note

This page is about **文語 / 古語 (classical Japanese)**—the historical written standard used for pre-modern texts and much formal writing—**not** modern Standard Japanese (現代語) unless the resource explicitly teaches classical forms.

## Grammars

- [Introduction to Classical Japanese (Bungo I) — syllabus (David Lurie, Columbia)](http://www.columbia.edu/~dbl11/Lurie-Bungo11-syll.pdf)
  - Notes: Course syllabus with a clear scope definition of Classical Japanese (bungo/kogo) and a structured outline of grammar topics and readings.
  - Format: PDF

- [Bungo Nyūmon (文語入門): A Brief Introduction to Classical Japanese](https://sengokudaimyo.com/bungo/introduction)
  - Notes: Free online introduction with sections on orthography, verbs/adjectives, particles, and common pitfalls. Written from a “learn it actively” perspective; useful as a reference alongside more formal materials.
  - Format: Web

- [NIHONGO eな: “Bungo Nyūmon” portal entry](https://nihongo-e-na.com/eng/site/id73.html)
  - Notes: Summary page describing the Bungo Nyūmon sections (incl. vocabulary problems and inflection/paradigms) and linking to the resource.
  - Format: Web

## Charts & Aids

- [Orthography (rekishiteki kanazukai) notes](https://sengokudaimyo.com/bungo/orthography)
  - Notes: Explains historical spelling conventions and common classical/modern spelling differences (e.g., ゐ/ゑ, けふ→きょう).
  - Format: Web

- [Vocabulary “false friends” and common pitfalls](https://sengokudaimyo.com/bungo/vocabulary-problems)
  - Notes: Examples where the same kanji/roots can mislead modern Japanese readers (classical vs modern meanings).
  - Format: Web

- [Verb paradigms (classical conjugation patterns)](https://sengokudaimyo.com/bungo/verbs)
  - Notes: Verb class overview with paradigm tables and example forms (useful as an inflection reference).
  - Format: Web

## Texts

- [Aozora Bunko (青空文庫)](https://www.aozora.gr.jp/)

  - Format: Web
