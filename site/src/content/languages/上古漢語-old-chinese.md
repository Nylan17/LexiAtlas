## Dictionaries

- [An overview of Chinese dictionaries](https://en.wikipedia.org/wiki/Chinese_dictionary)
  - Notes: Chinese has a very long history of dictionaries; if you're interested in the language, it will be useful to get a feel for what is actually out there, both print sources and otherwise.

- [Old Chinese Reconstructions](https://en.wiktionary.org/wiki/Appendix:Baxter-Sagart_Old_Chinese_reconstruction)
  - Notes: Lots of words with information about the character and derivation from older forms of Chinese.

- [ABC Etymological Dictionary of Old Chinese](https://muse.jhu.edu/books/9780824861339)
  - Format: PDF
  - Notes: A book available in PDF form by chapter.

- [Chinese Etymology](https://web.archive.org/web/20230401165837/http://www.internationalscientific.org/CharacterEtymology.aspx?characterInput=車&submitButton1=Etymology)
  - Notes: Kind of an older site, but it has some useful information about word histories.

- [Zhongwen Database](https://www.zhongwen.com/)
  - Notes: Includes a dictionary, texts, and resources to learn the language as well.

- [Digital Etymological Dictionary of Old Chinese](https://edoc.uchicago.edu/edoc_login/edoc_login.php)
  - Notes: Guest access is free--the learning curve on this one is a bit steeper than some of the other dictionaries on this page, though.

- [Old Chinese Basic Vocabulary Database](https://web.archive.org/web/20230401165837/http://language.psy.auckland.ac.nz/austronesian/language.php?id=331)
  - Notes: Has a small list of common or useful words.

- [Radical/Strokes Look-up](https://www.mdbg.net/chindict/chindict.php?page=radicals&cdqraddm=1)
  - Notes: The strokes needed to create a character, or "radicals," are an important measure for classifying the pieces of the written language.

- [Chinese Text Project Character Lookup](https://ctext.org/dictionary.pl?if=en)
  - Notes: By radicals here as well.

- [ZDic](https://www.zdic.net/hans/%E7%9B%AE)
  - Notes: Will only be useful if you already know Chinese, although it does show you how to draw the characters.

- [Chinese Characters](https://www.chinese-characters.org/radical/)
  - Notes: Also search by radical here; has information about ancient forms and variants, plus other stuff.

- [Yellowbridge Dictionary](https://www.yellowbridge.com/chinese/character-dictionary.php)
  - Notes: Shows information about the etymology as well.

## Grammars

- [Introduction to Literary Chinese](https://bloomington.iu.edu/)
  - Format: PDF
  - Notes: A very nice PDF file with exercises and other useful tools.

- [Introduction to Classical Chinese](https://chinesenotes.com/classical_chinese.php)
  - Notes: A website with various grammatical topics.

- [Chinese Script](https://web.archive.org/web/20230401180603/http://www.ancientscripts.com/chinese.html)
  - Notes: Information about the older ways that it was written.

- [A Concise Grammar of Classical Chinese](https://www.invisiblebooks.com/)
  - Notes: All on a single webpage.

- [Elements of Chinese Grammar](https://archive.org/details/elementschinese00confgoog)
  - Notes: Like the language it describes, this book is quite ancient.

- [Chinese: An Essential Grammar](https://web.archive.org/web/20230401180603/http://gjc.swu.edu.cn/UserFiles/File/20125311147206509.pdf)
  - Format: PDF
  - Notes: For the modern version, but it's free.

- [Zhongwen Database](https://www.zhongwen.com/)
  - Notes: Includes a dictionary, texts, and resources to learn the language as well.
